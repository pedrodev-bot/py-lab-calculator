# Python Calculator

## Overview
A application to calculate data. This tool was developed to solve complex calculations for university physics laboratory reports. This project is an improvemente of C++ Uncertainty Calculator, my previous command-line program. My goal is to expand my knowledge and try Python to create a application that users can use by themselves.

## The Impact
* **Efficiency:** This code reduced what was originally **days of manual calculations** for me and my colleagues into a **few hours of work**.
* **Precision:** Designed to handle calculations with high numerical precision to meet strict academic requirements.
* **Collaboration:** Used as a shared utility to help my lab group streamline data processing and focus on the physics analysis rather than arithmetic.
* **User Friendly:** Good UI to make sure users have a good experience.

## Status
This repository is currently up-to-date with my local development version. I intend to provide further updates, to make sure we have multiple features to laboratory needs. The code remains here as a reference for anyone tackling similar lab work.
